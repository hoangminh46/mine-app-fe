# 📝 Cấu Trúc CV Chi Tiết Cho Mid-Level Developer (3 năm KN)

> Hướng dẫn viết từng section trong CV, kèm ví dụ cụ thể.

---

## 1. Header (Thông Tin Cá Nhân)

### Bắt buộc có:
- **Họ tên** (rõ ràng, font lớn)
- **Số điện thoại** (có mã vùng nếu apply quốc tế: +84)
- **Email chuyên nghiệp** (firstname.lastname@gmail.com, KHÔNG dùng email "cute")
- **GitHub/GitLab link** ⭐ (cực kỳ quan trọng với dev)
- **LinkedIn profile**

### Nên có:
- **Portfolio website** (nếu có)
- **Thành phố** (không cần địa chỉ chi tiết)

### KHÔNG nên có:
- ❌ Ảnh chân dung (trừ khi JD yêu cầu hoặc apply công ty Nhật/Hàn)
- ❌ Ngày tháng năm sinh
- ❌ Tình trạng hôn nhân
- ❌ CMND/CCCD

### Ví dụ:
```
NGUYEN VAN A
Ho Chi Minh City, Vietnam | +84 912 345 678
nguyenvana.dev@gmail.com
github.com/nguyenvana | linkedin.com/in/nguyenvana
```

## 2. Professional Summary (Tóm Tắt Chuyên Môn)

> 2-3 câu cô đọng. Đây là "pitch" đầu tiên để recruiter quyết định đọc tiếp hay không.

### Công thức:
```
[Chức danh] with [X] years of experience in [core technologies].
[Thành tựu nổi bật nhất với con số cụ thể].
[Định hướng/giá trị mang lại].
```

### Ví dụ cho 3 năm KN:

**Frontend Developer:**
```
Frontend Engineer with 3 years of experience specializing in React.js, Next.js, 
and TypeScript. Delivered 5+ production-grade SaaS applications, improving page 
load performance by 40% through code splitting and lazy loading optimization. 
Passionate about building scalable, accessible UI and mentoring junior developers.
```

**Backend Developer:**
```
Backend Software Engineer with 3 years of experience building scalable microservices 
using Node.js, NestJS, and PostgreSQL. Reduced API response time by 35% through 
Redis caching and query optimization. Experienced in CI/CD pipelines, Docker 
containerization, and Agile methodologies.
```

**Fullstack Developer:**
```
Fullstack Developer with 3 years of experience in React, Node.js, and cloud services 
(AWS). Successfully led the development of an end-to-end CRM system serving 10,000+ 
users. Proven ability to own features from design to deployment, with a focus on 
clean code and automated testing.
```

### ⚠️ Tránh:
- ❌ "Seeking a challenging role..." (quá generic)
- ❌ "Hard-working and passionate..." (không có bằng chứng)
- ❌ Liệt kê quá nhiều công nghệ trong summary

---

## 3. Technical Skills (Kỹ Năng Kỹ Thuật)

> Phân nhóm rõ ràng, chỉ liệt kê 10-12 skills có thể TỰ TIN phỏng vấn.

### Format đề xuất:

```
TECHNICAL SKILLS
─────────────────────────────────────────────────────
Languages:      JavaScript, TypeScript, Python, SQL
Frameworks:     React.js, Next.js, NestJS, Express.js
Databases:      PostgreSQL, MongoDB, Redis
Cloud/DevOps:   AWS (EC2, S3, Lambda), Docker, GitHub Actions
Tools:          Git, Jira, Figma, Postman
Methodologies:  Agile/Scrum, CI/CD, Code Review
```

### Quy tắc:
- ✅ Chỉ liệt kê những gì bạn tự tin trả lời phỏng vấn
- ✅ Đặt skills phù hợp nhất với JD lên đầu mỗi nhóm
- ❌ KHÔNG tự đánh giá mức độ (VD: "Java: 80%" - rất chủ quan)
- ❌ KHÔNG liệt kê skills "rác" chỉ biết sơ qua
- ❌ KHÔNG dùng progress bar hoặc star rating

## 4. Work Experience (Kinh Nghiệm Làm Việc)

> Phần QUAN TRỌNG NHẤT của CV. Sử dụng thứ tự thời gian ngược (mới nhất lên đầu).

### Công thức cho mỗi bullet point:
```
[Action Verb] + [Task/Technology] + [Context] → [Measurable Result]
```

### Action Verbs mạnh cho dev:
| Level | Verbs |
|-------|-------|
| **Build/Create** | Engineered, Developed, Implemented, Built, Designed |
| **Improve** | Optimized, Refactored, Enhanced, Streamlined, Accelerated |
| **Lead** | Spearheaded, Led, Architected, Mentored, Coordinated |
| **Deliver** | Deployed, Shipped, Migrated, Automated, Integrated |

### Ví dụ cho 3 năm KN:

```
SENIOR FRONTEND DEVELOPER | ABC Tech JSC          Mar 2024 – Present
──────────────────────────────────────────────────────────────────────
• Architected a multi-tenant SaaS dashboard using React 18 and TypeScript, 
  serving 50+ enterprise clients with role-based access control.

• Optimized critical rendering path through code splitting and React.lazy(), 
  reducing initial bundle size by 45% and improving LCP from 3.2s to 1.4s.

• Implemented end-to-end testing suite with Cypress, achieving 85% code 
  coverage and reducing production bugs by 30%.

• Mentored 2 junior developers through structured code reviews and pair 
  programming sessions, accelerating their onboarding by 3 weeks.

• Collaborated with UX designers and backend engineers using Figma and 
  Swagger to deliver pixel-perfect, API-integrated features on schedule.
```

```
FRONTEND DEVELOPER | XYZ Solutions                 Jun 2022 – Feb 2024
──────────────────────────────────────────────────────────────────────
• Developed 3 customer-facing web applications using React.js and Redux, 
  collectively serving 20,000+ monthly active users.

• Integrated RESTful APIs with Axios and implemented client-side caching, 
  reducing redundant API calls by 60%.

• Built a reusable component library with Storybook documentation, adopted 
  by 4 development teams across the organization.

• Participated in Agile ceremonies (daily standups, sprint planning, 
  retrospectives) within a cross-functional team of 8 members.
```

### Điểm nhấn cho Mid-level (3 năm KN):
Ở level này, nhà tuyển dụng muốn thấy dấu hiệu "growth":
- ✅ **Collaboration:** Làm việc với product/design/backend teams
- ✅ **Mentorship:** Code review, hướng dẫn junior
- ✅ **System Ownership:** Tham gia quyết định kiến trúc, xử lý production issues
- ✅ **Independence:** Tự nhận ownership feature từ design đến deploy

## 5. Projects (Dự Án Nổi Bật)

> Dành cho side projects hoặc open-source contributions thể hiện kỹ năng mà work experience chưa cover.

### Format:

```
PROJECT NAME | Tech Stack                              Live Demo | GitHub
─────────────────────────────────────────────────────────────────────────
Brief description of the project, the problem it solves, 
and your role/contribution. Include metrics if possible.
```

### Ví dụ:
```
AI Chat Platform | Next.js, NestJS, WebSocket, PostgreSQL
github.com/yourname/ai-chat | demo: chat.yoursite.com
─────────────────────────────────────────────────────────
Built a real-time AI-powered customer support platform handling 1,000+ 
concurrent connections. Integrated OpenAI API with streaming responses 
and implemented multi-tenant architecture with Zalo/Facebook Messenger 
channel support.
```

## 6. Education (Học Vấn)

> Ngắn gọn. Với 3 năm KN, section này không cần chiếm nhiều không gian.

```
B.Sc. in Computer Science | University of Technology (HCMUT)    2019 – 2023
GPA: 3.2/4.0 (chỉ ghi nếu >= 3.0)
```

## 7. Certifications (Chứng Chỉ)

> Chỉ liệt kê chứng chỉ có giá trị và còn hiệu lực.

```
• AWS Certified Solutions Architect – Associate        2025
• Meta Frontend Developer Professional Certificate     2024
```

### Chứng chỉ có giá trị cao trong thị trường VN:
- AWS / GCP / Azure certifications
- Kubernetes (CKA/CKAD)
- Meta/Google Professional Certificates (Coursera)
- TOEIC/IELTS (đặc biệt quan trọng ở VN)

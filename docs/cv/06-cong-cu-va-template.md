## 2. Chi Tiết Từng Công Cụ

### 🏆 Overleaf (LaTeX) - Gold Standard
**Website:** overleaf.com

**Ưu điểm:**
- Typography cực kỳ chuyên nghiệp, "crisp"
- ATS-friendly tuyệt đối
- Version control tích hợp (Git)
- Thể hiện technical sophistication
- Nhiều template chất lượng cao

**Nhược điểm:**
- Learning curve cao nếu chưa biết LaTeX
- Chỉnh sửa nhanh khó hơn so với WYSIWYG tools

**Templates đề xuất:**
- `Jake's Resume` - Phổ biến nhất cho dev
- `Deedy-Resume` - Modern, 2 cột (cẩn thận ATS)
- `AltaCV` - Clean, 1 cột

### 🏆 SWEResume.app - Dành riêng cho SWE
**Website:** sweresume.app

**Ưu điểm:**
- Thiết kế riêng cho Software Engineer
- ATS-safe, single-column
- Gợi ý viết bullet points với metrics
- Export PDF chất lượng cao

### ⭐ Google Docs - Đơn giản & Hiệu quả
**Ưu điểm:**
- Miễn phí hoàn toàn
- Easy collaboration & sharing
- Export PDF dễ dàng
- Nhiều template miễn phí

**Template tìm kiếm:** "Software Engineer Resume Template Google Docs"

### ⚠️ Canva - KHÔNG khuyên cho Dev
**Lý do tránh:**
- Layout phức tạp → ATS KHÔNG đọc được
- Icons, graphics → Bị ATS bỏ qua
- 2-column layout → Thứ tự đọc bị đảo lộn
- **Chỉ phù hợp cho vị trí Design/Creative**

## 3. Công Cụ Hỗ Trợ Khác

### Kiểm tra ATS Score:
| Công cụ | Chức năng | Link |
|---------|----------|------|
| **Jobscan** | So sánh CV với JD, chấm điểm match | jobscan.co |
| **Resume Worded** | Phân tích ATS score + gợi ý cải thiện | resumeworded.com |
| **TopCV Score** | Phù hợp thị trường VN | topcv.vn |

### Grammar & Writing:
| Công cụ | Chức năng |

| **Hemingway Editor** | Đơn giản hóa câu văn |

## 4. Workflow Đề Xuất

```
1. Chọn tool: Overleaf hoặc SWEResume.app
       ↓
2. Viết nội dung theo cấu trúc ở file 02
       ↓
3. Tailor keywords theo JD cụ thể
       ↓
4. Test ATS: Paste vào Notepad + dùng Jobscan
       ↓
5. Review grammar: Grammarly + đọc lại
       ↓
6. Export PDF (< 1MB, text-selectable)
       ↓
7. Nộp + gửi email trực tiếp cho recruiter
```
